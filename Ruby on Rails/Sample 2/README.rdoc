== Ruby on Rails Sample Project 2

This sample code uses scaffolding to create a sample application with two models (Users and Posts). 

This sample uses: 

- Ruby version 2.1.2
- Rails version 4.2.4
- TinyTDS (http://github.com/rails-sqlserver/tiny_tds) version 0.7.0 
- activerecord-sqlserver-adapter (http://github.com/rails-sqlserver/activerecord-sqlserver-adapter) version 4.2.4
